宽度：{Dimensions.get('window').width}

7、在安卓中设置textInput的underlineColorAndroid string
文本框的下划线颜色(译注：如果要去掉文本框的边框，请将此属性设为透明transparent)。即
underlineColorAndroid=”transparent”
8、经常出现，this.refs.scroll.scrollTo…(this.refs.scroll is not undefined)原因就真的是undefined，如果用到了this，考虑执行的函数中是不是this的作用于不对，如果是函数的话，要记得在函数后面.bind(this)；
9、若有ListView无法显示的情况，尽管对ListView设置了height，flex=1都没有用，请注意是否设置了ListView的父View的flex和height，比如

在此处已经吃了好几次亏，找了一晚上的bug才找出来，谨记谨记
10、在自定义的组件里面慎用flex = 1，特别是小组件，否则会出现排班错乱的情况， 比如citymedia中的
MFHorizontalCell中的cellContainer中用了flex = 1
11、在写XX的react 版本中，在写单纯的订单列表组件时，如果在listView的外层再嵌套一个scrollView，则会使listView的
onEndReached方法不执行
12、在使用phonegap调用系统相册、相机发现是英文的系统相簿界面后标题显示“photos”，但是手机语言已经设置显示中文，纠结半天，最终在info.plist设置解决问题，我反正闲着也是没事于是就搜索了下相关的资料,发现在项目的info.plist里面添加Localized resources can be mixed YES（表示是否允许应用程序获取框架库内语言）即可解决这个问题。特此记录下以便以后查看和别人解决问题
13、有的插件需要link到原生项目中，比如react-native-image-picker安装后只需添加一句react-native link即可,
