Angular 日期本地化

Angular 日期本地化，以“angular-datepicker”为例

（1）安装“angular-datepicker”，bower install angular-datepicker —save，

其中“—save”表示在安装这个插件的时候会把这个插件所依赖的其他插件也安装进去，比如这个插件依赖bootstrap，那么它就会把bootstrap也安装，并且别人下载了你的bower.json文件时，也会在依赖中出现，别人bower install的时候，会自动安装这个依赖。

（2）其次，基本上很多的datepicker插件都依赖于moment.js，

所以你还要安装moment，即 bower install moment —save 或者 npm install moment --save

（3）把相关的js文件引入到HTML中：

index.html

请关注“zh-cn.js”，其为你要本地化成的语言，如不引入的话将为“en”，即英语显示；然后把datepicker指令注入到你的angular App中，以“angular-datepicker”为例

（4）需要一个angular本地化的js文件，名曰：angular-locale_zh-hans-cn.js，然后引入到你的项目中。
